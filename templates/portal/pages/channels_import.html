Formato M3U

El archivo M3U debe seguir el formato estandar:

#EXTM3U
#EXTINF:-1 tvg-id="ID" tvg-name="Nombre" tvg-logo="URL_LOGO" group-title="Categoria",Nombre Canal
http://url-del-stream.m3u8
Atributos soportados:
  • tvg-id: ID para EPG
  • tvg-name: Nombre alternativo
  • tvg-logo: URL del logo
  • group-title: Categoria
Comportamiento:
  • Canales existentes se actualizan
  • Canales nuevos se crean
  • Categorias se crean automaticamente
